제안서 MD 변환 도구 설치 가이드

설치만 차근차근 따라 하세요. 사용은 파일을 넣고 변환 버튼을 누르면 끝입니다.

1. ZIP 파일 압축 풀기

  1. 전달받은 ZIP 파일을 PC에 저장합니다.
  2. ZIP 파일을 마우스 오른쪽 버튼으로 클릭합니다.
  3. 압축 풀기를 선택해 폴더로 풉니다.
ZIP 파일 안에서 실행.bat를 바로 실행하면 안 됩니다. 반드시 압축을 먼저 풀어 주세요.

2. 폴더 위치 정하기

압축을 푼 제안서 MD 변환 폴더는 아래 위치 중 하나에 두는 것을 권장합니다.

C:\Users\사용자명\Desktop\제안서 MD 변환
C:\Users\사용자명\Documents\제안서 MD 변환
C:\Tools\제안서 MD 변환
C:\Program Files, C:\Windows, 읽기 전용 USB, 네트워크 드라이브, 압축파일 내부는 피하세요.

3. 실행하기

  1. 압축을 푼 폴더를 엽니다.
  2. 실행.bat를 더블클릭합니다.
  3. 브라우저가 열리면 준비 완료입니다.
브라우저가 자동으로 열리지 않으면 주소창에 http://localhost:3000을 입력하세요.

4. 사용하기

  1. 파일을 화면에 끌어다 놓거나 클릭해서 선택합니다.
  2. 변환 시작을 누릅니다.
  3. 완료된 파일을 다운로드합니다.

여러 파일을 한 번에 선택할 수 있고, 완료된 파일을 체크해서 선택 다운로드도 할 수 있습니다.

.hwpx
.hwp
.pdf
.pptx

5. 종료하기

브라우저만 닫으면 서버가 계속 실행 중일 수 있습니다.

완전히 종료하려면 작업표시줄의 Proposal MD Converter - Server 창을 닫아 주세요.

문제가 생기면

브라우저가 안 열립니다
주소창에 http://localhost:3000을 직접 입력해 보세요.
변경사항이 반영되지 않습니다
기존 서버가 실행 중일 수 있습니다. 작업표시줄의 Proposal MD Converter - Server 창을 닫고 실행.bat를 다시 실행하세요.
실행 창이 바로 꺼집니다
폴더 위치 문제일 수 있습니다. Desktop 또는 Documents로 옮긴 뒤 다시 실행하세요.
node.exe 오류가 나옵니다
runtime\node.exe가 빠진 상태입니다. ZIP 파일을 다시 받아 전체 압축을 풀어 주세요.
여러 파일 다운로드가 막힙니다
브라우저가 자동 다운로드를 막은 것입니다. 주소창 근처의 다운로드 허용 안내를 확인해 주세요.

공유할 때 확인할 것

  • uploads, output 안의 기존 파일은 비워도 됩니다.
  • 이 HTML 파일과 사용법.md도 함께 넣어 주세요.
